# Задача 10 — проверка IPv4 / IPv6

## Участник группы
Шуваев Илья

## Описание решения
Для проверки корректности IP‑адресов используется модуль `ipaddress`, который занимается основной валидацией.  
Дополнительно применяются регулярные выражения для предварительной фильтрации форматов адресов.  
Функция `normalize_ipv6` приводит IPv6‑адрес к компактному виду: убирает лишние нули и сокращает `::`.

## Как запустить
```bash
python solution.py
```

## Ввод
**Примеры IPv4:**  
`192.168.0.1`  
`10.0.0.1`

**Примеры IPv6:**  
`2001:0db8:0000:0000:0000:ff00:0042:8329`  
`2001:db8::1`

## Вывод
```
Тип: IPv6
Нормализованный IPv6: 2001:db8::1
```
