# Задача 2 — строки, слова и самое длинное слово

# Участник группы: Шуваев Илья

## Описание решения
Для корректного разделения слов используется регулярное выражение  
`re.split(r"[^\w'\-]+")`, которое разбивает текст по всем символам,  
не являющимся буквами, цифрами, апострофами или дефисами.

Программа:
- считывает файл построчно;
- подсчитывает количество строк и слов;
- отслеживает самое длинное слово и его длину.

## Как запустить
```bash
python solution.py
```

## Ввод
Путь к файлу:  
`C:\Users\User\Desktop\test.txt`

## Вывод
```
Lines: 9
Words: 20
Longest word: sdfsdgdfg
Length: 9
```
