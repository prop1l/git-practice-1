# Задача 7 — Частота символов

# Участник группы: Шуваев Илья

## Описание решения
Используется `collections.Counter` для подсчёта частоты каждого символа.  
При необходимости регистр можно игнорировать, переведя текст в нижний регистр.  
Итоговый словарь сортируется по убыванию частоты.

## Как запустить
```bash
python solution.py
```

## Ввод
Путь к файлу:  
`C:\Users\User\Desktop\test.txt`

## Вывод
```
's': 18
'd': 17
'f': 15
' ': 14
'
': 9
'g': 7
'h': 1
'S': 1
'A': 1
'F': 1
```
